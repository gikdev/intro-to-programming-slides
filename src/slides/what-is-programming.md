# برنامه‌نویسی چیه؟

- **برنامه‌نویس**: کسی هستش که اون برنامه رو می‌نویسه، به کامپیوتر میگه چیکار بکنه.
- **زبان برنامه‌نویسی**: همون زبان مشترک بین انسان و کامپیوتر هست.
